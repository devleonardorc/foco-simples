Um organizador de tarefas simples para estudos, trabalho e vida pessoal.

## O que o projeto faz

- Adiciona tarefas com categoria e prazo.
- Marca tarefas como concluídas.
- Exclui tarefas.
- Filtra por todas, hoje, semana, pendentes e concluídas.
- Mostra progresso, total de tarefas, concluídas e atrasadas.
- Salva os dados no navegador usando `localStorage`.
- Tem modo claro e modo escuro.

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Como abrir

Abra o arquivo `index.html` no navegador.
